Inicio — Crear un reporte
1. Toca una categoría: HVAC, Bombas, Motores o Control.
2. Selecciona el tipo de formulario (Mantenimiento o Procedimiento).
3. Llena los datos del equipo, checklist, mediciones y cierre.
4. Toca 💾 Guardar reporte y asígnalo a un cliente y servicio/COT.
5. Descarga el reporte en 📄 Word o 🖨 PDF.
📋
Reportes — Clientes y servicios
Clientes → Toca un cliente para ver sus servicios/COTs.
Servicio/COT → Toca para ver todos los reportes de ese servicio.
+ Nuevo reporte → Desde la COT, crea un reporte con datos pre-llenados.
📁 Mover → Abre un reporte y toca Mover para cambiarlo de COT.
🗑 Eliminar → Aparece solo si la carpeta está vacía (clientes/COTs).
🔍 Buscador → Busca por TAG, técnico, OT, cliente o tipo de equipo.
🤖
Diagnóstico IA
1. Llena las mediciones del equipo (antes y después).
2. Toca 🤖 Diagnóstico IA.
3. La IA analiza los datos y genera hallazgos técnicos automáticamente.
4. Revisa y edita los hallazgos antes de guardar.
Requiere conexión a internet.
💡
Tips útiles
Técnico guardado: La app recuerda tu nombre en este dispositivo.
Datos pre-llenados: Al crear desde una COT, el cliente y OT se llenan solos.
Instalar la app: En Chrome, toca el menú ⋮ → "Agregar a pantalla de inicio".
Sin internet: Puedes llenar el formulario, pero guarda cuando tengas conexión.
Actualizar: Toca 🔄 en la pantalla de Reportes para ver cambios de otros técnicos.
